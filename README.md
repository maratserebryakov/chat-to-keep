# chat-to-keep
keep the whole routerai.ru chat greed with AI Model to the Google Keep app in shape of a link to your gist.github.com
therefor you need to get your gist token to use this chrome based browser extension (Yandex browser et cetera)
How to install
Create a chat-to-keep folder and put all the files there.
Save any 128x128 image as icon128.png
Chrome → chrome://extensions → turn on "Developer Mode" → "Download unpacked" → select a folder
Create a GitHub token once via the link in the popup(scope gist only)
Open a chat on RouterAI → click the extension icon → "Save chat in Keep"
Result
A note appears in Google Keep with a short link to GitHub Gist
Follow the link to a beautifully rendered Markdown with code highlighting, headers, lists, and tables.
Gist secret (not indexed, but available via the link)
